Chapter 8 Stiff ODEs

m-files:

  • decay1.m Sample code using matlab ODE solver for decay process.
  • decaytest.m Test decay1 for various tolerances.
  • plotS.m Plot the region of absolute stability S (and also the order star if desired) for a one-step method.
  • makeplotS.m Sets R(z) for a specific one-step method and calls plotS. Contains R(z) for several methods.
  • plotSrkc.m Plot stability region S for Runge-Kutta-Chebyshev methods.
  • chebpoly.m Evaluate Chebyshev polynomial (used by plotSrkc.m)
  • chebpoly1.m Evaluate first derivative of Chebyshev polynomial (used by plotSrkc.m)
  • chebpoly2.m Evaluate second derivative of Chebyshev polynomial (used by plotSrkc.m)
